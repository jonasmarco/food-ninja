This project was made with:

  * [Tailwind CSS](https://tailwindcss.com/docs/installation)
  * [Heroicons](https://heroicons.dev/)
  * And ❤️

## How to run?
 ```javascript
	git clone https://github.com/jonasmarco/food-ninja
	cd food-ninja
	npm i
	npm run build-css
	open public/index.html on browser
	it's done!
```
